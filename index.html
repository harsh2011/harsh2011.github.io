Text Semantic Similarity Open Source

A model to measure the semantic similarity between sentences. Central to this approach is a corpus-based unsupervised learning model based on Second-Order Pointwise Mutual Information. The model relies on commonality of neighbouring trigrams along with TF/IDF factor to build a statistical measure to calculate the semantic similarity score. The syntactic similarity score is calculated using a combined subsequence matching measure.
